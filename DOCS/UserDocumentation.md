User Documentation
Starting the Application

Open the project in your IDE.

Run MenuApp.java.

Follow the on-screen instructions.

Menu Options Explained
1. Add Patient

Adds a new patient to the system.

2. Add Doctor

Adds a new doctor with specialization.

3. Add Medication

Adds a new medication with stock and expiry date.

4. Create Prescription

Creates a prescription linking doctor, patient, and medication.

Reduces medication stock automatically.

Prevents creation if stock is zero.

5. Full Report

Displays all doctors, patients, medications, and prescriptions.

6. Check Expired Medications

Displays medications past their expiry date.

7. Restock Medications

Adds quantity to all medications in stock.

8. Search

Searches patient, doctor, or medication by name.

9. Edit

Allows updating:

Patient phone number

Doctor specialization

10. Delete

Allows deleting:

Patient

Doctor

Medication

11. Doctor Prescriptions

Displays all prescriptions issued by a specific doctor.

12. Past Year Drug Summary

Shows medications prescribed within the last 12 months.

13. Exit

Closes the application.

Data Storage

All data is stored in memory during runtime.
Data is not persisted after application closes.
